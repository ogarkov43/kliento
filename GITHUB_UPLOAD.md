# Инструкция по загрузке на GitHub

## Файлы для загрузки:

Загрузите ВСЕ эти файлы и папки на GitHub:

### Файлы в корне:
- `package.json`
- `package-lock.json`
- `tsconfig.json`
- `tsconfig.node.json`
- `vite.config.ts`
- `tailwind.config.js`
- `postcss.config.js`
- `vercel.json`
- `netlify.toml`
- `_redirects`
- `index.html`
- `.gitignore`
- `README.md`

### Папки:
- `src/` (вся папка со всеми файлами)
- `.vscode/` (если есть)

### НЕ загружайте:
- ❌ `node_modules/` 
- ❌ `dist/`
- ❌ `.git/`
- ❌ `*.log`
- ❌ `*.pid`

## Быстрый способ:

1. Откройте папку: `/Users/kirillogarkov/Desktop/kliento`
2. Выделите все файлы (Cmd+A)
3. Исключите папки `node_modules` и `dist`
4. Перетащите на GitHub
