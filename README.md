# Padel App - Telegram Mini App

Приложение для отслеживания игр в падел.

## Установка

```bash
npm run install:all
```

## Запуск

```bash
npm run dev
```

Это запустит одновременно frontend (порт 3000) и backend (порт 3001).

## Структура проекта

- `frontend/` - React + TypeScript приложение
- `backend/` - Node.js + Express API
