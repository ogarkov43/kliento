# Открыть в Cursor

Приложение запущено на http://localhost:5173

**Чтобы открыть в Cursor:**

1. Нажмите **Cmd+Shift+P** (или **Ctrl+Shift+P**)
2. Введите: `Simple Browser: Show`
3. В открывшемся поле введите: `http://localhost:5173`
4. Нажмите Enter

Или:

1. Откройте панель **Ports** (Cmd+Shift+P → "Ports: Focus on Ports View")
2. Найдите порт **5173**
3. Нажмите на иконку открытия в браузере
